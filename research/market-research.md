## 3. PRD 작성 초보자를 위한 도구

### 3-1. 초보자 전용/친화적 도구

| 도구 | 초보자 지원 방식 |
|------|-----------------|
| **PRD Pal (Emergent)** | 아이디어 → 구조화된 PRD + 비주얼 로드맵으로 변환. AI 코파일럿 방식 |
| **AI PRD Creator (3-Step Wizard)** | 3단계 마법사 인터페이스: (1) 입력 (AI 프리필 또는 수동) → (2) 생성 → (3) 검토/내보내기 |
| **BeeWeb AI PRD Generator** | 음성 설명 → PRD 자동 변환. 기술 지식 불필요. 5-7분 완성 |
| **Vibe PRD Generator (vibegen.net)** | 한국어 지원. 단계별 가이드. 5-10분 내 완성. 무료. 회원가입 불필요 |
| **QuillBot AI PRD Generator** | 템플릿/포맷팅 없이 바로 생성. 간단한 입력 |

### 3-2. 가이드/튜토리얼 제공 도구

| 도구/서비스 | 교육 콘텐츠 |
|------------|------------|
| **Product School** | PRD 작성 가이드 + 완전한 예시 제공 |
| **Perforce** | PRD 작성 완전 가이드 (How to Write a PRD) |
| **Udemy** | "Mastering PRD: Step-by-Step Guide for Product Manager" 유료 코스 |
| **PMPrompt** | AI를 활용한 PRD 작성 단계별 가이드 + 예시 (2025) |
| **HelloPM** | PRD 작성법 튜토리얼 |
| **Chisel Blog** | AI 활용 PRD 작성 팁 & 기법 |

### 3-3. 초보자가 겪는 주요 어려움

1. **"좋은 PRD"의 기준을 모름** - 참고할 수 있는 예시 부족
2. **품질에 대한 불안감** - 경력이 적을수록 PRD 품질에 대한 두려움이 큼
3. **구조 설계의 어려움** - 어떤 섹션을 포함해야 하는지 모름
4. **시간 부족** - 포괄적인 PRD 작성에 2-3시간 소요
5. **다양한 템플릿으로 인한 혼란** - 어떤 형식을 따라야 하는지 판단 어려움
6. **데이터 기반 작성의 어려움** - 의견 기반 문서가 되기 쉬움

## 5. 경쟁 분석을 통한 기회

### 5-1. 기존 도구들의 부족한 점

| 문제 영역 | 상세 |
|-----------|------|
| **한국어 지원 부재** | ChatPRD, Rock-n-Roll, Chisel 등 주요 도구가 영어 전용. 한국어 PRD 생성 품질이 낮음 |
| **초보자 가이드 부족** | 대부분의 도구가 "입력하면 생성"하는 방식이지, 왜 이 섹션이 필요한지 교육하지 않음 |
| **컨텍스트 부족** | AI가 회사 맥락, 팀 상황, 산업 특성을 반영하지 못함 |
| **도구 파편화** | 고객 인터뷰, 피드백, 지원 티켓이 각각 다른 도구에 분산되어 PRD에 통합이 어려움 |
| **일회성 생성** | 대부분 초안 생성에만 집중. 반복적 개선/업데이트 지원 부족 |
| **데이터 연결 부족** | 의견 기반 PRD가 되기 쉬움. 실제 데이터/인사이트 연결 부족 |
| **과도한 길이** | 생성된 PRD가 너무 길어서 아무도 읽지 않는 문제 |

### 5-2. 초보자 관점에서의 진입 장벽

| 장벽 | 상세 |
|------|------|
| **PRD 구조 이해 부족** | 어떤 섹션이 필요하고, 각 섹션에 무엇을 써야 하는지 모름 |
| **"좋은 PRD" 기준 부재** | 완성된 좋은 PRD 예시를 볼 기회가 적음 |
| **전문 용어의 벽** | acceptance criteria, user story, edge case 등 용어에 익숙하지 않음 |
| **품질 불안감** | 작성한 PRD가 충분히 좋은지 판단할 수 없음 |
| **가격 장벽** | 유료 도구들의 월 구독료가 주니어 PM/학생에게 부담 |
| **영어 중심** | 가이드/튜토리얼이 대부분 영어로 되어 있음 |

### 5-3. 한국어 사용자를 위한 차별화 포인트

#### 핵심 기회 영역

1. **한국어 네이티브 AI PRD 생성기**
   - 한국어로 자연스러운 PRD 생성 (단순 번역이 아닌 한국 비즈니스 맥락 반영)
   - 한국 스타트업/IT 기업의 PRD 관행 반영
   - 1Pager, 6Pager 등 한국에서 선호하는 문서 형식 지원

2. **교육형 PRD 작성 도구 (Learn-as-you-write)**
   - 각 섹션별 "왜 이것이 필요한가" 설명
   - 한국어 예시와 가이드 제공
   - 단계별 인터랙티브 작성 지원
   - PRD 품질 점수 및 개선 제안

3. **바이브 코딩 연계**
   - PRD → Cursor/Claude Code/Bolt 등 바이브 코딩 도구 연동
   - PRD에서 바로 프로토타입 생성
   - 한국 개발자 생태계와의 통합

4. **합리적 가격 정책**
   - 주니어 PM/학생 대상 무료 또는 저가 플랜
   - 한국 시장에 맞는 가격 설정

5. **한국 PM 커뮤니티 연계**
   - 한국어 PRD 템플릿 마켓플레이스
   - PM 커뮤니티 (요즘IT, 퍼블리, 커리어리) 연동
   - 선배 PM의 PRD 피드백/리뷰 기능

6. **Notion/Confluence 내보내기 최적화**
   - 한국 기업에서 주로 사용하는 Notion 형식으로 바로 내보내기
   - Jira 연동을 통한 개발팀 핸드오프

### 가장 큰 기회: "한국어 + 초보자 친화" 영역

현재 시장에서 **한국어를 네이티브로 지원하면서** 동시에 **초보자에게 교육적 가이드를 제공하는** PRD 도구는 사실상 존재하지 않음. Vibe PRD Generator가 한국어를 지원하지만 교육적 깊이는 제한적임.

### 핵심 경쟁 우위 제안

1. **"배우면서 쓰는 PRD" 컨셉** - 단순 생성이 아닌, PRD 작성 역량을 키워주는 도구
2. **한국어 최적화** - 한국 비즈니스 문화와 PM 관행을 반영한 PRD 구조
3. **바이브 코딩 시대의 PRD** - 코드 생성 AI와 직접 연결되는 PRD 포맷
4. **합리적 가격** - 주니어 PM이 부담 없이 시작할 수 있는 가격 구조
